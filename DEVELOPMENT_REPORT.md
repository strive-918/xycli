## Architecture

```
CLI (Commander.js)
  → Agent Loop (observe → plan → act → reflect)
    → Anthropic Provider (chat + streamChat)
    → Tool Registry
      → file_read / file_write / terminal_exec
    → Session Store (JSON files)
```

## Interfaces (Extension Points)

- `ITool` — add tools without modifying core (DESIGN.md §4)
- `IProvider` — swap LLMs without modifying core (DESIGN.md §6)
- `ToolRegistry` — register built-in, MCP, or plugin tools uniformly
- `SessionStore` — replace JSON with SQLite in M4
